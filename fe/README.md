# Attendance System - Frontend

A modern React frontend built with Vite and Tailwind CSS for the Attendance Management System.

## Features

- User Management (Create, View, Delete)
- Attendance Capture via QR Code
- Modern UI with Tailwind CSS
- Fast development with Vite

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Backend Connection

Make sure the backend is running on `http://localhost:5000` before using the frontend.
